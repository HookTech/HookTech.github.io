# Hexo 图片目录结构说明

## 标准目录结构

### 1. 全局图片目录
- `source/images/` - 存放全站通用的图片
- 引用方式：`![alt](/images/image.jpg)`

### 2. 文章专用图片目录
- `source/_posts/文章名/` - 每篇文章的专用图片目录
- 引用方式：`![alt](/年/月/日/文章名/image.jpg)`

### 3. 草稿图片目录
- `source/_drafts/文章名/` - 草稿文章的图片目录
- 引用方式：`![alt](/年/月/日/文章名/image.jpg)`

## 最佳实践

1. **全局图片**：网站logo、favicon、通用图标等放在 `source/images/`
2. **文章图片**：文章内容相关的图片建议放在文章同名目录下
3. **图片格式**：推荐使用 JPG、PNG、WebP 等格式
4. **图片优化**：建议压缩图片以提高加载速度

## 使用示例

```markdown
<!-- 全局图片 -->
![网站Logo](/images/logo.png)

<!-- 文章图片 -->
![文章配图](/年/月/日/文章名/image.jpg)
```
